Brainstorm Notebook

A Chrome extension that generates interactive mind-map graphs from your AI conversations on Claude, ChatGPT, and Gemini.

Install from Chrome Web Store
View on GitHub

What It Does

Each conversation turn becomes a node in an interactive mind-map, with edges showing how topics flow and connect across your discussion. Brainstorm Notebook helps you visualize the structure of your AI conversations in real time.

Features

  • Auto-generated mind-maps from conversation turns
  • Real-time updates as you chat
  • Batch rebuild for entire conversations
  • Interactive canvas with pan, zoom, and drag
  • Manual edge editing (connect or cut)
  • Encrypted API key storage (AES-256-GCM)
  • Per-conversation graph persistence

Supported Platforms

Claude (claude.ai)
ChatGPT (chatgpt.com)
Gemini (gemini.google.com)

Supported API Providers

Anthropic (Claude)
OpenAI (GPT)
Google (Gemini)
